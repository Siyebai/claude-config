# .claude/ 内部结构说明

> 每个目录的职责、加载时机、缓存影响

## rules/ — 全局规则（始终加载）

启动时自动注入系统提示。保持精简稳定是缓存命中的核心。

| 文件 | 行数 | 职责 | 变更频率 |
|------|------|------|----------|
| craft.md | ~20 | 代码工艺标准 | 极少 |
| identity.md | ~10 | 身份与行为 | 极少 |
| ops.md | ~10 | 运维与清理 | 极少 |

## skills/ — 可调用技能（按需加载）

通过 `/skill-name` 或 `Skill` 工具调用时才加载完整 SKILL.md。
目录名对应技能名，每个目录包含一个 `SKILL.md`。

当前: 16 个技能（从 448 精简），覆盖日常开发场景。

## agents/ — 子代理定义（按需加载）

通过 `Agent` 工具 `subagent_type` 参数调用时才加载对应的 `.md`。
自定义 prompt 覆盖内置默认，提供更精准的行为指令。

当前: 7 个代理（从 89 精简），覆盖架构/审查/修复/Git。

## hooks/ — 会话钩子（自动触发）

| 钩子 | 触发时机 | 动作 |
|------|----------|------|
| session-start.ps1 | 每次会话启动 | 清理 paste-cache/shell-snapshots/telemetry |
| session-end.ps1 | 每次会话结束 | 压缩 >500KB 的会话文件到 backups/ |

## commands/ — 内置命令（Claude Code 自带）

斜杠命令实现文件。由 Claude Code 安装和管理，勿手动修改。

## projects/.../memory/ — 持久记忆（按需检索）

跨会话记忆系统。MEMORY.md 索引始终加载（保持静态），具体记忆文件通过向量检索按需加载。

**规则**: 记忆存文件，不存缓存。文件永久，会话归零。

## docs/ — 系统文档（参考）

Codex 中继、服务守护、系统架构等参考文档。不参与运行时加载。
