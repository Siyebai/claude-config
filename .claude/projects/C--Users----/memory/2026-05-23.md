### 5. 构建 2 个新工具 + 引擎 v3.1

```
~/novels/tools/
├── style-checker/         # 新建 — 四维风格参数 + 语言指纹偏差检测
│   └── index.js           # profile / check / diff / batch (330行)
└── knowledge-graph/       # 新建 — 70+关系 + Union-Find + 图愈合
    └── index.js           # build / relations / query / heal / conflicts / export (540行)
```

**style-checker 核心能力:**
- 句长分析 (avg/median/p25/p75/std/分布) + 句长偏差检测
- 对话占比分析 (4种引号模式 + 直接对话行)
- 弱化副词密度 (15个高频词) + "地"字结构副词
- 段落节奏 (扩展/收缩/稳定模式检测)
- 五感分布 (5类关键词库 → 主导感官 + 多样性)
- 比喻密度 + 判断句密度
- 六维偏差检测: sentence_length / dialogue_ratio / modifier_density / paragraph_rhythm / sensory_primary / metaphor_density
- batch 模式: 全书风格一致性检查

**knowledge-graph 核心能力:**
- 70+ 中文关系类型规范化 (`RELATION_NORM`) + 6 大类染色 (`RELATION_CATEGORY`)
- Union-Find 别名消解 (含安全过滤 — 亲属词/头衔/集合词不进 UF)
- 图愈合: 临时节点毕业 (置信度≥0.75) + 重复合并 (相似度≥0.75候选/≥0.85自动) + 孤立清理 (≥3章无引用)
- 关系冲突检测: family_vs_hostile (CRITICAL) / intimate_vs_hostile (HIGH)
- 多源构建: character_bible + character_state + relationship_matrix + 章节文本 + facts.jsonl
- 导出: JSON / DOT (Graphviz) / Cytoscape.js
- 图谱存储: `03_plot/knowledge_graph.json`
