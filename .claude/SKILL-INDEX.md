# 调度规则（优先级由高到低）

```
用户指令
  ↓
① 关键词匹配第一层核心技能 → 自动调用（多个匹配则组合）
  ↓
② 未匹配则扫描第二层辅助技能 → 按场景调用
  ↓
③ 仍不匹配则第三层上下文技能 → 被动触发
  ↓
④ 无任何匹配 → 常规能力处理
```

## 组合调度规则

当任务涉及多个领域时, 组合调用:

| 任务类型 | 自动组合 |
|---------|---------|
| **做一个新页面** | design-taste-frontend + react-patterns + motion-ui + coding-standards |
| **审计并修复UI** | redesign-existing-projects + frontend-a11y + high-end-visual-design |
| **性能优化** | react-performance + benchmark + cost-tracking + strategic-compact |
| **安全审查** | security-scan + security-bounty-hunter + coding-standards |
| **Agent系统搭建** | autonomous-agent-harness + agent-architecture-audit + enterprise-agent-ops |
