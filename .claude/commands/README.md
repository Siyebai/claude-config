# Claude 命令套件

## 命名空间

| 命名空间 | 说明 | 文件数 |
|---------|------|-------|
| `/dev:*` | 开发工具：调试/解释/重构/构建/监控 | 14 |
| `/docs:*` | 文档生成：API文档 | 1 |
| `/performance:*` | 性能审计 | 1 |
| `/project:*` | 项目管理：特性/包/健康检查 | 5 |
| `/security:*` | 安全审计 | 1 |
| `/setup:*` | 环境搭建：数据库/API/migration | 7 |
| `/skills:*` | 技能管理：审计/优化/建议 | 3 |
| `/test:*` | 测试：用例生成/编写 | 2 |
| 根级别 | 通用：review/plan/commit/pr/audit/help/session | 14 |

## 用法
`/<command>` 或 `/<namespace>:<command>`
