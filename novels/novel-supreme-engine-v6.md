## 一、风格宪法

### 叙事体质

```
核心体质: 热叙事底 + 口语魂
  ├── 叙事层: 热 — 允许直接写情绪，但用身体反应和物质细节承载
  ├── 对话层: 口语 — 像人说话，不像作家写字
  ├── 切换能力: 关键场景切换为冷叙事
  └── 底线: 不煽情、不俗套、不空洞

感官优先级: 体感 > 视觉 > 听觉
语言质地: 口语但保留文学性。每句写完在心里默念——不顺口就改。
```

### 冷热切换 — 五级温度模型

最痛的场景用最冷的方法写。哭不出来比放声大哭更有力。

| 级别 | 名称 | 句长 | 叙事距离 | 情绪暴露度 | 使用场景 |
|------|------|------|---------|-----------|---------|
| +2 | 滚烫 | 15-28字 | 极近 | 100% | 正向高潮(胜利/重逢/狂喜)、情绪完全释放 |
| +1 | 温热 | 12-22字 | 近 | 60% | 日常互动、内心反思、温馨、幽默 |
| 0 | 常温 | 10-20字 | 中 | 30% | 信息传递、过渡、环境建立 |
| -1 | 微冷 | 8-16字 | 远 | 10% | 紧张/对峙/悬疑/压抑愤怒 |
| -2 | 极寒 | 6-12字 | 极远 | 0% | 负向高潮(死亡/告别/背叛/重大失去) |

**切换规则**:
- 正向高潮 → +2。负向高潮 → -2。越痛越冷
- 禁止: +2→+2连续(情感透支)、-2→+2(冰火两重天)、0→+2(无铺垫跳跃)
- 推荐: 快档章 0→+1→+2→-1→-2→0。中档章 +1→0→+1→0→-1→+1

**+2滚烫技法**: 允许内心独白(口语化)、感官密度≥3种、身体反应写具体。禁止空洞情绪词堆砌
**-2极寒技法**: 全部简单动词(站/看/走/摸/放)、零情绪词零评论、对话标签只说"说"、用感官缺席写情感在场、重复一个无意义动作

### 五条铁律

1. **禁止AI高频词**: 心中一惊/眼中闪过/不由得/似乎/仿佛/微微一笑/沉声道/淡淡道/冷声/眸光/唇角/身形
2. **对话标签只用**: 说/问/答/喊/叫。偶尔动作打断(笑了一下/沉默了)。绝对禁止X道
3. **情绪不空洞**: "她很伤心"❌ → "她下颌收紧。指节发白。嗓子里堵着什么。" ✅。身体反应≥2种
4. **每句话默念**: 不顺口的改。写得顺口比写得好看重要十倍
5. **遮名可辨**: 遮住名字能分清谁在说话 → 通过

## 三、写作流水线 Phase 0-9

### Phase 0: 灵感孵化 (~30min)
**输入**: 灵感碎片 → **输出**: `idea_seed.md`
1. 一句话种子: "当[主角]遭遇[事件]，必须[行动]，否则[灾难]"
2. 欲望三角: Want / Obstacle / Cost / Need
3. 逆推三锚点: 终局→中点→开场画面
4. 五要素: 目标读者/风格/禁区/自动化等级/规模

### Phase 1: 人物塑魂 (~60min)
每个有名字的角色填写设计卡:

```yaml
角色名: / 功能位: 主角/反派/Ally/Mentor/Shadow/Foil/Love/Shapeshifter
# 外在: 外貌关键词(≤5) / 标志习惯 / 标志物品
# 内在: Want / Need / Flaw(阻碍得到Need的东西) / Ghost(造就Flaw的源头)
# 声音: 说话风格 / 口癖(1-2) / 禁用词 / 典型句例(3句)
# 情绪出口(不是"他生气了"——生气时具体做什么):
#   愤怒/恐惧/悲伤/开心/撒谎时各有不同的身体反应和行为
# 底牌: 最怕/最想要/愿付代价/不能跨的底线
# 矛盾: 内在矛盾(两个互相打架的特质) / 外在矛盾
```

**配角功能位**: Ally(支持但必须有不同点)/Mentor(必须有自己的盲区)/Shadow(主角的另一种可能——最强配角类型)/Foil(同一情境不同反应)/Shapeshifter(忠诚不确定)/Love(必须有独立于主角的目标)

**自检**: Want≠Need有张力? 反派有"认为自己才是对的"理由? 遮名可辨? 配角功能位明确?

### Phase 2: 大纲规划 (~60min)
`novel beat <N> --markdown` → beat_sheet.md。为每章标注: 节奏档/温度/POV/伏笔操作。预设≥5条伏笔。

### Phase 3: 写前SOP (~15min/章)

**A. 上下文加载**: 角色状态刷新 + 距离衰减事实检索(≤3章×1.0/4-10章×0.7/>10章×0.5) + 伏笔扫描(未回收>已回收×1.5→优先回收) + 世界规则刷新

**B. 风格锚定**: 确定温度(+2到-2) + 冷场景`novel mode generate yu-hua` + 情感高潮确认切换点 + 朗读风格金样本500字

**C. 场景澄清5问**: POV的Want? 具体障碍? 情况如何变化? 读者获什么新信息(≥1条)? 情感弧线?

**D. 机械免疫预检**: 高危场景预标注 + 禁止词预查

**E-F. 概要(200-500字)→Architect审查→写作授权签发**

### Phase 4: 快写 (~60-90min/章)
**导演心法**: 搭场景→放角色进去→观察→记录。不是你写，是角色自己动。

硬约束: 对话=行动(推动/揭示/冲突)。遮名可辨。情绪不空洞(身体反应+物质细节)。句长可变(不连续5句同长)。对话标签只说/问/答

### Phase 5: 热叙事三箭 (~30min/章)

**第一箭·情绪强化**: 每个情感节点→身体反应≥2种? 大悲/大痛→冷叙事(句长6-12字/简单动词/零度标签/感官缺席写在场)

**第二箭·口语检测**: 能念出来吗? 每句话有"说话的人"吗? 夹带作者文采了吗? 标签是"说/问/答"吗?

**第三箭·多角色差异化**: 遮名可辨? 同事件各角色反应不同? 各有情绪出口? 配角有独立目标?

### Phase 6: 增色 (~45min/章)
- 感官补全: 每场景≥1种非视觉感官
- 物质细节: 每300字≥1个具体物质(有叙事功能)
- 节奏: 长段(≥200字)后用短段(≤30字)切断。对话中插入非对话打断
- 自检: 对话有冲突? 动作句长≤15? 心理太多情绪词→转身体反应? 环境有叙事功能?

### Phase 7: 精修 (~30min/章)
1. 开头第一句够不够好? (不铺垫不描写)
2. 结尾制造翻页欲望了吗?
3. 有可删的段落吗? → 删
4. 读一遍——哪段无聊? → 删或注入冲突

### Phase 8: 质量门禁 (~10min/章)
```bash
novel deslop scan ch_N.md     # CRITICAL=0 ∧ HIGH≤2
novel style lint ch_N.md      # HIGH≤3
novel mode apply <mode> ch_N.md  # CRITICAL=0 (如指定风格)
```
→ 全部PASS → 下一章。不满足 → 回到Phase 5

### Phase 9: 里程碑体检 (每10章)
```bash
novel score milestone <dir>       # Q-Score ≥8.0
novel entropy batch <dir>         # ETC≤2.0 ∧ EWC≤1.5
novel entropy audit <dir>         # 知识过滤
novel graph heal <dir>            # 图愈合
novel style lint-batch <dir>      # 全书文笔
```

## 五、角色语音系统

### 遮名可辨的7个维度

| 维度 | 含义 | 示例对比 |
|------|------|---------|
| 句长偏好 | 说话用多长的句子 | A:"走。" B:"我觉得我们可以考虑先离开。" |
| 词汇层级 | 雅/俗/文/野/专业 | A:"吃饭" B:"用膳" C:"干饭" |
| 句式习惯 | 陈述/疑问/反问/祈使偏好 | A:全陈述 B:三句一反问 |
| 节奏模式 | 连绵还是间断 | A:一口气说完 B:说一句停一下 |
| 口头禅 | 重复出现的词汇或句式 | A:"你听我说" B:"是吧"结尾 |
| 回避语言 | 绝不说的话 | A:绝不说"爱" B:绝不说"对不起" |
| 对话策略 | 直来直去/绕弯子 | A:要什么直接说 B:从不直接说 |

### 同情绪不同表达

| 情绪 | 外放型 | 内敛型 | 转移型 | 计算型 |
|------|--------|--------|--------|--------|
| 愤怒 | 脸红/青筋/脏话/砸东西 | 下颌收紧/话变短/语气冷 | 笑着说的但笑不是笑/喝酒/运动 | 表面无变化/语气变客气/开始计划 |
| 恐惧 | 出汗/发抖/音量变大 | 手心出汗/不说话/嘴唇干 | 话变多语速快/反而往前走 | 身体僵硬/大脑空白/动不了 |
| 悲伤 | 哭/哽咽/说不完整句子 | 嗓子堵/话变多以免哭 | 不停做事/不让自己停 | 什么都不感觉/一个人待着 |
| 撒谎 | 眼神飘忽/解释过多/补细节 | 眼神反而更坚定/话变少 | 特别放松/开着玩笑撒谎 | 装傻/绕圈子/反问对方 |

### 硬规则
- 每角色一张语音卡。≥3维差异才算有效区分
- 每场对话遮名重读。两个角色可互换台词而察觉不到 = 失败
- 角色绝不在对话中说"我生气了""我很难过"
- 配角语音 ≠ 主角语音的降级版（是独立的声音）

## 六、质量体系

### Q-Score 八维 (权重)

①情节结构20% | ②角色塑造18% | ③文笔语言12% | ④世界观10% | ⑤主题思想8% | ⑥情感冲击12% | ⑦阅读体验12% | ⑧预期管理8%

```
Q-Score = Σ(维度得分 × 权重)。ETC≤2.0→情节≥7分。EWC≤1.5→世界观≥7分
PASS: 单章③⑥⑦≥7.0。里程碑Q-Score≥8.0
WARN: 有5.0-6.9。FAIL: 有<5.0
```

### 风险预警
连续下滑(任一维度30章↓)→暂停。情感疲乏(5章同情绪)→调节奏。伏笔堆积(未回收>已回收×1.5)→优先回收。角色闲置(配角10章未出场)

### 反AI味检测 v2.0

**12类检测维度**:

| # | 类别 | 阈值/章 | 说明 |
|---|------|---------|------|
| 1 | AI高频词 | ≤5/3000字 | 86→150+词。扩展: 眼底/眸色/唇角/指尖/身形 |
| 2 | 弱化副词 | ≤3/千字 | 微微/淡淡/缓缓/轻轻/悄悄 + 新增"算是/某种程度上/简直" |
| 3 | 意义膨胀 | 命中即标记 | 意义深远/前所未有/叹为观止/不容忽视 |
| 4 | 结论套话 | 命中即标记 | 展望未来/未来可期/总而言之 |
| 5 | 论文式段落 | ≤25%段 | 段落以总结句开头 |
| 6 | 正式语体 | 命中即标记 | 于是乎/一方面/另一方面/从而/因而 |
| 7 | 排比三连 | ≤2处 | X、Y、Z结构。新增检测: "X的YX的YX的Y" |
| 8 | "是"字泛滥 | ≤15%/千字句 | E-Prime中文版。连续3句含"是"→WARN |
| 9 | AI句式模板 | 零容忍 | 结尾"然而等待着他的是..."/开头"随着..."/"心中涌起一股..." |
| 10 | 段落结构AI味 | 均匀度<15% | 连续5段长度偏差<15%→机械。连续3段同主语开头→单调 |
| 11 | 解释型写作 | 极严格 | 行动后紧跟解释意义→删解释。情感场景后紧跟"一种X感包围了他"→删 |
| 12 | 对话AI味 | — | 对话中传递设定/所有角色语法完整无打断/回应模式化 |

**高潮场景专门规则** (最严格):
- 连续3句含情绪词 → CRITICAL
- "他终于明白/意识到/知道" → CRITICAL
- 感叹号>1处 → WARN
- "泪"字>1次 → HIGH
- 对话标签只能是"说/问"

## 七、场景写作速查

### 四类场景

```
对话场景: 潜台词(说≠想)/权力关系(话量不对称)/对话=行动/插入打断/遮名
动作场景: 句长≤15/感官加速(视听触同时)/动作揭示角色/节奏快→缓冲→快
心理场景: 身体反应/物质投射/冷热切换点/情绪不直说
环境场景: 功能优先(无叙事=删)/空间透视(宏观→焦点→反应)/环境=心理/≥2种感官
```

### 高危场景禁止
情感: 直说情绪词。打斗: 长句/复杂比喻。对话: 纯信息交换。揭秘: 一次性全揭示。过渡: >300字
