## 一、风格宪法

写之前先锁定"我是谁"。一个写作者不能什么风格都写。

### 叙事体质

```
核心体质: 热叙事底 + 口语魂
  ├── 叙事层: 热 (允许直接写情绪，但用身体反应和物质细节承载)
  ├── 对话层: 口语 (像人说话，不像作家写字)
  ├── 切换能力: 关键场景切换为冷叙事 (情感高潮处"退一步写")
  └── 底线: 不煽情、不俗套、不空洞

感官优先级: 体感 > 视觉 > 听觉
语言质地: 口语但保留文学性。每句写完在心里默念——不顺口就改。
```

### 冷热切换规则

最痛的场景用最冷的方法写。哭不出来比放声大哭更有力。

| 情境 | 温度 | 写法 |
|------|------|------|
| 日常互动/对话 | 热 | 直接写情绪感受，对话有温度 |
| 情感高潮 (死亡/告别/背叛) | **冷** | 余华模式——退一步。外部动作+感官细节。越痛越克制 |
| 紧张/悬疑 | 冷 | 短句、零评论、动作驱动 |
| 角色内心反思 | 热 | 口语化内心独白。允许"我想""我觉得" |
| 环境建立/过渡 | 温 | 中句长，适量描写，不轰炸 |
| 对话冲突 | 热口语 | 有火药味或潜台词，不写成辩论赛 |

### 五条铁律

1. **禁止AI高频词**: 心中一惊/眼中闪过/不由得/似乎/仿佛/微微一笑/沉声道/淡淡道/冷声/目光/身形/冷笑
2. **对话标签只用**: 说/问/答/喊/叫。偶尔允许动作打断(笑了一下/沉默了)，绝对禁止X道(淡淡道/沉声道)
3. **情绪不空洞**: "她很伤心"❌ → "她下颌收紧。指节发白。嗓子里有什么东西堵着。" ✅
4. **每句话默念**: 不顺口的改。写得顺口比写得好看重要十倍。
5. **对话里没有作者**: 遮住名字能分清谁在说话 → 通过。

## 三、写作流水线

### Phase 0: 灵感孵化 (~30min)

**输入**: 一个灵感碎片 → **输出**: `idea_seed.md`

1. 一句话种子: "当[主角]遭遇[事件]，必须[行动]，否则[灾难]；同时[更大危机]正在发酵。"
2. 欲望三角: Want / Obstacle / Cost / Need
3. 逆推三锚点: 终局画面 → 中点画面 → 开场画面
4. 五要素确认: 目标读者 / 写作风格 / 核心禁区 / 自动化等级 / 目标规模

### Phase 1: 人物塑魂 (~60min)

**输入**: idea_seed.md → **输出**: character_bible.md + character_state.md

每个有名字的角色填写完整设计卡:

```yaml
角色名:
功能位: 主角 / 反派 / Ally / Mentor / Shadow / Foil / Love / Shapeshifter

# 外在标识
外貌关键词 (≤5个):
标志性动作或习惯:
标志性物品:

# 内在驱动力
Want (表层欲望):
Need (深层需求):
Flaw (缺陷):               # 阻碍他得到 Need 的东西
Ghost (创伤/过去):          # 造就 Flaw 的源头事件

# 声音设计
说话风格:                   # 话多/话少/文绉绉/大老粗/毒舌/木讷
口癖 (1-2个):
禁用词:
典型句例 (3句):

# 情绪出口 (不是"他生气了"——他生气时具体做什么)
愤怒时: / 恐惧时: / 悲伤时: / 开心时: / 撒谎时:

# 底牌
最怕什么: / 最想要什么: / 愿意付出什么代价: / 不能跨越的底线:

# 矛盾
内在矛盾:                   # 两个互相打架的特质
外在矛盾:                   # 与环境/他人的核心冲突
```

**配角功能位**: 每个配角有一个明确功能。Ally(盟友)/Mentor(导师)/Shadow(代表主角另一种可能)/Foil(对照凸显主角)/Shapeshifter(忠诚不确定)/Love(必须有独立目标)。

**自检**:
- 主角的 Want 和 Need 之间有张力吗？
- 反派有"为什么认为自己才是对的"的理由吗？
- 遮住名字，每个角色的对话能分清吗？
- 每个配角的"功能位"明确吗？

### Phase 2: 大纲规划 (~60min)

```
novel beat <N> --markdown → beat_sheet.md + plot_threads.md
```

为每章标注: 节奏档(快/中/慢) / 风格温度(热/温/冷) / POV角色 / 伏笔操作(回收/新埋)。预设至少5条伏笔。

### Phase 3: 写前 SOP (~15min/章)

**A. 上下文加载**
- 角色状态刷新 — 出场角色最新状态
- 事实摘要检索 — 距离衰减加权 Top-20 (≤3章×1.0, 4-10章×0.7, >10章×0.5)
- 伏笔扫描 — 本章应回收/应新埋。未回收 > 已回收×1.5 → 优先回收
- 世界规则刷新 — 最近5章规则变更

**B. 风格锚定**
- 确定本章温度 (热/温/冷)
- 冷场景 → `novel mode generate yu-hua`
- 情感高潮 → 确认冷热切换点
- 朗读风格金样本500字 (不浏览，是朗读)

**C. 场景澄清 — 5个强制问题**
1. POV角色的Want是什么？(具体到这3000字)
2. 阻止他的障碍是什么？(不能只是"敌人太强")
3. 本章结束时，情况如何变化？(不能和开头一样)
4. 读者获得什么新信息？(至少1条。不能是纯过渡章)
5. 本章的情感弧线是什么？(开头什么情绪→结尾什么情绪)

**D. 机械免疫预检**
- 高危场景预标注: [情感场景]禁止直说情绪词 / [打斗场景]禁止长句复杂比喻 / [对话场景]禁止纯信息交换 / [揭秘场景]禁止一次性全揭示 / [过渡场景]禁止超300字
- 禁止词预查: 标记可能触发的AI高频词

**E. 概要写作** (200-500字) → Architect审查

**F. 写作授权签发**
```
chapter: N | style_temp: hot/warm/cold | style_mode: (if cold) yu-hua
→ Scribe: 开始写正文。严格按照风格模式指令执行。
```

### Phase 4: 快写 (~60-90min/章)

**导演心法**: 你不是"作家"，你是"导演"。在脑子里搭建场景，把角色放进去，观察他们，记录他们做什么说什么。

**硬约束**:
- 对话即行动 — 每句话推动情节/揭示角色/制造冲突，三者必居其一
- 遮名可辨 — 写完每段对话遮住名字读一遍
- 情绪不空洞 — 用身体反应+物质细节
- 句长可变 — 不要连续5句长度相近
- 对话标签只用: 说/问/答/喊/叫
- 禁止AI高频词

### Phase 5: 热叙事三箭 (~30min/章)

**第一箭·情绪强化**: 每个情感节点→情绪够具体？身体反应≥2种？大悲/大痛→切换到冷叙事(句长10-18字/简单动词/零度对话标签/感官缺席写情感在场)

**第二箭·口语检测**: 所有对话→能念出来吗？每句话有"说话的人"吗？对话中夹带作者文采了吗？对话标签是"说/问/答"吗？

**第三箭·多角色差异化**: 遮住名字→分清谁在说话？同一事件→每个角色反应不同？每个角色有自己情绪出口？配角有独立目标吗？

### Phase 6: 增色 (~45min/章)

- 感官补全: 每场景补充≥1种非视觉感官 (触觉/听觉/嗅觉)
- 物质细节: 每300字≥1个具体物质。每个细节有叙事功能
- 节奏微调: 长段落(≥200字)后用短段落(≤30字)切断。连续3句同主语→变化。对话中插入非对话打断
- 场景自查: 对话→是否有冲突潜台词？动作→句长≤15字？心理→是否太多情绪词？环境→是否有叙事功能？

### Phase 7: 精修 (~30min/章)

**四问**:
1. 开头第一句够不够好？不铺垫不描写——直接切入
2. 结尾收得住吗？制造了翻页的欲望吗？
3. 有没有哪段删掉故事不受影响？→ 删
4. 读一遍——哪段无聊？→ 删或注入冲突

### Phase 8: 质量门禁 (~10min/章)

```bash
novel deslop scan chapter_N.md    # CRITICAL=0 ∧ HIGH=0
novel style lint chapter_N.md     # HIGH ≤ 3
novel mode apply <mode> chapter_N.md  # CRITICAL=0 (如指定了风格模式)
```

→ 全部 PASS → 进入下一章。不满足 → 回到 Phase 5。

### Phase 9: 里程碑体检 (每10章)

```bash
novel score milestone <dir>       # 全8维 Q-Score。目标 ≥8.0
novel entropy batch <dir>         # ETC≤2.0 ∧ EWC≤1.5
novel entropy audit <dir>         # 三层知识过滤
novel graph heal <dir>            # 图愈合
novel style lint-batch <dir>      # 全书文笔健康
```

## 五、质量体系

### Q-Score 八维评分

| 维度 | 权重 | 类型 |
|------|------|------|
| ① 情节与结构 | 20% | 客观 — ETC映射 |
| ② 角色塑造 | 18% | 客观 |
| ③ 文笔与语言 | 12% | 客观 — deslop/风格检测映射 |
| ④ 世界观与设定 | 10% | 客观 — EWC映射 |
| ⑤ 主题与思想 | 8% | 客观 |
| ⑥ 情感冲击 | 12% | 主观 |
| ⑦ 阅读体验 | 12% | 主观 |
| ⑧ 预期管理 | 8% | 主观 |

```
Q-Score = Σ(维度得分 × 维度权重)
ETC ≤ 2.0 → 情节连贯性 ≥ 7分
EWC ≤ 1.5 → 世界观一致性 ≥ 7分
```

### 门禁标准

| 级别 | 单章门禁 | 里程碑(10章) |
|------|---------|------------|
| PASS | ③⑥⑦ ≥ 7.0 | Q-Score ≥ 8.0 |
| WARN | 有 5.0-6.9 | 任一维度连续下滑 |
| FAIL | 有 < 5.0 | Q-Score < 6.0 |

### 风险预警

| 信号 | 阈值 | 动作 |
|------|------|------|
| 连续下滑 | 任一维度连续30章下降 | 暂停→根因分析 |
| 情感疲乏 | 连续5章同一情绪 | 调整节奏 |
| 伏笔堆积 | 未回收 > 已回收×1.5 | 优先回收 |
| 角色闲置 | 功能位配角10章未出场 | 规划出场 |

## 六、场景写作速查

### 四类场景指令

```
对话场景:
  □ 潜台词 — 说的≠想的。每段对话有第三层意思
  □ 权力关系 — 话量不对称反映谁在上风
  □ 对话=行动 — 不是交换信息，是推动/揭示/冲突
  □ 打断 — 沉默/动作/环境细节插入在对话中间
  □ 遮名自检

动作场景:
  □ 句长收缩 ≤15字。一动作=一句。不解释
  □ 感官加速 — 视觉+听觉+触觉同时
  □ 动作揭示角色 — 不是描述他是什么人，是看他怎么做
  □ 节奏变化 — 快(短句)→释放(长句缓冲)→快

心理场景:
  □ 身体反应 — 认知失调→生理外显
  □ 物质投射 — 角色注意到的环境细节=心理状态
  □ 冷热切换点 — 最深心理痛用最冷方法写
  □ 情绪不直说

环境场景:
  □ 功能优先 — 每个细节有叙事功能。没有=删除
  □ 空间透视 — 宏观→异常焦点→角色反应
  □ 环境=心理 — 光线温度=心理温度
  □ 感官组合 — 至少两种感官同时写
```

### 高危场景禁止

| 类型 | 禁止 |
|------|------|
| 情感场景 | 直说情绪词 |
| 打斗场景 | 长句/复杂比喻/过多描写 |
| 对话场景 | 纯信息交换式对话 |
| 揭秘场景 | 一次性全部揭示 |
| 过渡场景 | 超过300字 |
