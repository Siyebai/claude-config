> 夜间进化产出 2026-05-23。v1.0: 7类86短语 → v2.0: 12类200+短语 + 句式模式 + 段落结构

## 一、新增检测维度

### 第8类: "是"字判断句泛滥 (E-Prime中文版)

AI写作大量使用"X是Y"的判断句式，导致文笔僵硬。

**检测规则**:
- 连续3句含"是"字判断 → WARN
- 每千字"是"字判断句 > 15% → HIGH
- 情感/心理场景中"是"字判断 → 逐句标记

```
❌ "那是一栋很旧的房子。墙是灰的。院子里是杂草。"
✅ "房子旧了。灰墙。院子里杂草齐膝。"
```

### 第9类: AI句式模板

AI生成的句式高度模板化，以下是高频模板：

**开头模板** (AI喜欢在段落开头用的):
- "随着..." → 标记 "随着夜幕降临/随着时间的推移/随着事态的发展"
- "在...之下/之中" → "在月光的照耀下/在众人的注视下/在恐惧的笼罩下"
- "当...的时候" → "当他转身的时候/当她说出那句话的时候"
- "那一刻..." → "那一刻他终于明白/那一刻她的心碎了"
- "从此..." → "从此他再也不.../从此一切都变了"
- "就在此时..." → 机械的转折

**结尾模板** (AI喜欢在段落/章节结尾用的):
- "但他/她不知道的是..."
- "然而等待着他/她的却是..."
- "这一切才刚刚开始..."
- "而这一切不过是个开始"
- "谁也不会想到..."
- "而这仅仅是序幕"

**情绪模板** (AI表达情绪时的套路):
- "心中涌起一股..." (涌起暖流/涌起不安/涌起恐惧)
- "一股...涌上心头"
- "不知为何..." (不知为何感到/不知为何想起)
- "仿佛...一般" (仿佛被什么扼住一般/仿佛坠入冰窟一般)
- "说不清是什么..." (说不清是什么感觉/说不清是什么滋味)
- "他的眼神中闪过..." (闪过复杂/闪过慌乱/闪过惊愕/闪过决然)
- "他的眼底掠过..." (掠过笑意/掠过阴霾/掠过暗芒)
- "嘴角勾起一抹..." (冷笑/弧度/嘲讽的弧度)

**动作模板** (AI写动作时的套路):
- "缓缓..." (缓缓抬起/缓缓睁开/缓缓转身/缓缓闭上) — 标记"缓缓"频率
- "微微..." (微微一怔/微微挑眉/微微颔首/微微侧头)
- "轻轻..." (轻轻拍了拍/轻轻叹了口气/轻轻摇了摇头)
- "深深..." (深深看了一眼/深深鞠了一躬/深深吸了一口气)
- "一把..." (一把抓住/一把抱住/一把拉住/一把扯开)
- "X了X..." (闭了闭眼/摇了摇头/抿了抿唇/攥了攥拳) — 此结构每千字≤2处
- "垂眸/抬眸/敛眸" — AI最爱的眼部动作，每千字≤1处
- "眸光/目光" — 每千字≤1处
- "身形一..." (身形一顿/身形一晃/身形一闪) — 每章≤1处

**感官模板**:
- "一股...的气息" (一股危险的气息/一股熟悉的气息)
- "...在空中弥漫" (紧张的气氛在空中弥漫)
- "...的气氛" (诡异的气氛/压抑的气氛/沉闷的气氛)
- "...的感觉" (一种说不出的感觉/一种奇怪的感觉)

### 第10类: 段落结构AI味

AI生成的段落有高度规律的结构模式：

**三段式段落** (AI最爱):
一段开头句总括 + 中间展开 + 结尾句升华。每段都是小论文。
```
检测: 段落首句为抽象总结 + 末句为情感/意义升华 → WARN
```

**均匀段落长度**:
AI生成的段落长度高度均匀（每段3-5句，150-250字），缺乏呼吸感。
```
检测: 连续5段长度偏差<15% → HIGH (段落节奏过于均匀)
```

**段落开头单调**:
连续3段以同一主语开头（"他/她/它"）。
```
检测: 连续3段首句主语相同 → WARN
```

**对话段落模式化**:
```
❌ AI模式:
"对话内容。"角色A说道。
"对话内容。"角色B回应。
"对话内容。"角色A又说。
→ 每句对话都有标签，且标签位置固定（都在句末）

✅ 人类模式:
"对话内容。"
角色B没说话。
"对话内容！"
角色A站起来。他走到窗前，背对着她。"对话内容。"
→ 标签不规则分布（有时在句首、有时在句中、有时无标签）。穿插动作打断。
```

### 第11类: 信息密度异常

**解释型写作** (AI最大的问题——替读者理解):
```
检测: 角色做出一个行动后，下一句立即解释这个行动的意义
❌ "她转身离开了。她的这个决定意味着她放弃了最后的机会。"
✅ "她转身离开了。"
```

**情感标注** (AI喜欢给情感贴标签):
```
检测: 描写一个场景后，下一句标注这个场景的情感意义
❌ "他站在那里，看着空荡荡的房间。一种深深的孤独感包围了他。"
✅ "他站在那里，看着空荡荡的房间。窗帘在风里动了一下。"
```

### 第12类: 对话AI味

**对话中的作者声音** (角色变成了作者的传声筒):
```
检测特征:
- 角色在对话中说出大量抽象概念/哲理
- 对话中的信息密度远高于正常人说话
- 对话中没有打断、没有重复、没有"嗯""啊""那个"
- 所有角色的对话都语法完整
```

**对话回应模式化**:
```
AI最喜欢的对话节奏: A说一句→B准确地回应了A的意图→A再回应B
❌ "你来了。"她说。
   "我来了。"他说。
   "你不该来的。"
   "但我已经来了。"

这种看似有风格的对白已经被AI写烂了。
```

**对话中传递设定的AI味**:
```
检测: 对话中明显在向读者传递背景信息
❌ "你知道的，自从三年前那场大战之后，我们的族人就一直..."
✅ 读者通过上下文和细节自行推断（不通过角色之口直接告知）
```

## 二、现有规则升级

### 第1类升级: AI高频词 (86→150+)

**新增词汇** (按类别):

身体反应类:
```
眼底/眸中/眸色/眸光/眼帘/睫毛
唇角/嘴角/唇边/唇畔
指尖/指腹/掌心
身形/身影/身姿/身躯
面容/脸庞/脸颊/面庞
```

情感标记类:
```
心尖一颤/心口一紧/心往下沉
眼眶泛红/眼眶微红/眼眶湿润
鼻子一酸/喉头一紧/喉咙发紧
脊背发凉/背脊一凉/背后一凉
瞳孔一缩/瞳孔微缩/瞳孔地震
```

动作描写类:
```
转身离去/转身离开/转身就走
头也不回/头也不抬/头也不回地
唇角微勾/嘴角微扬/嘴角勾起
眸光微动/眸光闪烁/眸光暗了暗
眼底闪过/眼底掠过/眼底划过
```

五感描写类:
```
空气中弥漫着/弥漫着...的气息
一股...的味道/一股...的气味
...在空气中飘散
```

时间过渡类:
```
须臾/片刻/半晌/少顷/俄顷 (AI过度使用文言时间词)
良久/许久
转瞬/刹那/霎时/瞬间/眨眼间
```

### 第2类升级: 弱化副词扩展

新增需要检测的中文弱化结构:
```
"算是..." → 弱化判断
"某种程度上..." → 模糊限制
"某种意义上..." → 模糊限制  
"多多少少..." → 弱化
"或多或少..." → 弱化
"差不多..." → 模糊
"几乎..." → 弱化（每千字≤2处）
"简直..." → 过度强调（每千字≤1处）
```

### 第3类升级: 三联排比扩展

新增检测类型:
```
检测1: "X的Y，X的Y，X的Y" 结构（AI最爱）
  ❌ "他的眼神里有无助，有迷茫，有说不出的痛苦。"

检测2: "没有X，没有Y，只有Z" 结构
  ❌ "这里没有温暖，没有希望，只有无尽的黑暗。"

检测3: "X也好，Y也罢，Z也成" 结构
  ❌ "成功也好，失败也罢，平庸也成。"
```

## 三、场景化检测规则

不同场景有不同的AI味容忍度：

| 检测项 | 对话场景 | 动作场景 | 心理场景 | 环境场景 | 高潮场景 |
|--------|---------|---------|---------|---------|---------|
| AI高频词阈值/千字 | ≤3 | ≤5 | ≤8 | ≤5 | ≤2 |
| 弱化副词/千字 | ≤2 | ≤1 | ≤3 | ≤3 | ≤0 |
| "是"字判断/千字 | ≤10% | ≤8% | ≤12% | ≤15% | ≤5% |
| 三联排比/章 | ≤1 | ≤0 | ≤2 | ≤1 | ≤0 |
| 段落长度偏差 | 不检测 | ≤30% | ≤25% | ≤20% | 不检测 |
| 解释型写作 | 严格 | 严格 | 中等 | 中等 | 极严格 |
| 情感标注 | 极严格 | 严格 | 极严格 | 中等 | 极严格 |

### 高潮场景专门规则 (最严格)
```
高潮场景额外检测:
□ 连续3句包含情绪词 → CRITICAL (高潮处不能堆砌情绪)
□ 任何形式的"他终于明白/他终于意识到/他终于知道" → CRITICAL
□ "一切都..." (一切都结束了/一切都变了/一切都晚了) → CRITICAL
□ 感叹号使用 > 1处/高潮场景 → WARN
□ "泪"字出现 > 1次/高潮场景 → HIGH
□ 对话标签超过"说/问" → HIGH (高潮场景对话标签必须最简)
```

## 五、综合检测门禁 v2.0

```bash
novel deslop scan <file> --mode <master-id>

检测输出:
  CRITICAL (必须清零):
    - 高潮场景情绪词 > 0
    - 高潮场景解释型写作 > 0
    - 情感标注 (任何场景)
    - 风格模式禁止模式命中
    
  HIGH (目标=0, 允许≤2):
    - AI高频词 > 阈值
    - AI句式模板命中
    - 段落结构AI味
    - E-Prime "是"字泛滥
    
  MEDIUM (建议修复):
    - 弱化副词超标
    - 三联排比超标
    - 段落长度过于均匀
    - 对话标签位置模式化
    
  INFO:
    - 感官缺失提醒
    - 句长变异度低
    - 对话中信息密度异常
```
