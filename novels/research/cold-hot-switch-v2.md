# 冷热切换系统 v2.0

> 夜间进化产出 2026-05-23。v1.0: 三档温度(热/温/冷) → v2.0: 五级温度(-2到+2) + 渐变曲线 + 场景衔接规则

---

## 一、五级温度模型

v1.0 的"热/温/冷"三档在实际写作中不够精细。v2.0 扩展为五级：

| 级别 | 名称 | 一句描述 | 句长范围 | 叙事距离 | 情绪暴露度 |
|------|------|---------|---------|---------|-----------|
| +2 | 滚烫 | 情绪完全打开，感官全开 | 15-28字 | 极近(贴着角色皮肤写) | 100% — 允许直接写情绪 |
| +1 | 温热 | 情绪在场但克制表达 | 12-22字 | 近(角色的视角但不贴皮肤) | 60% — 情绪通过身体+少量直陈 |
| 0 | 常温 | 叙述者保持中立距离 | 10-20字 | 中(正常叙事距离) | 30% — 情绪只通过行动暗示 |
| -1 | 微冷 | 退一步写，用外部细节 | 8-16字 | 远(观察者距离) | 10% — 情绪严格外化 |
| -2 | 极寒 | 余华模式。句短到极限。零度 | 6-12字 | 极远(上帝视角但无情感) | 0% — 零情绪词。零评论 |

### 各级别详细参数

**+2 滚烫 (Hot-Intense)**
```
使用场景: 角色完全释放的时刻（大笑/大哭/暴怒/激情）
技法:
  - 允许写内心独白（口语化）
  - 允许直陈情绪（但不能只用情绪词）
  - 感官密度拉满（≥3种感官同时在场）
  - 身体反应写具体（不是"他很高兴"——是具体的身体反应）
禁止:
  - 空洞的情绪词堆砌
  - 情感＞行动（情绪必须有行为出口）
  - 在无铺垫的情况下突然+2
示例:
  "他大笑起来。笑得太厉害，腰都直不起来。一手扶着墙，一手捂着肚子。
   眼泪都笑出来了。也不知道在笑什么。就是想笑。停不下来。"
```

**+1 温热 (Warm)**
```
使用场景: 日常互动/角色内心反思/温馨场景/幽默段落
技法:
  - 情绪通过身体反应 + 物质细节承载
  - 对话有温度（角色之间的互动可见情感连接）
  - 允许少量情绪直陈（≤2处/千字）
  - 感官以体感+视觉为主
禁止:
  - 空洞的形容词："温馨""美好""感人"
  - 情绪直陈超过2处/千字
  - 煽情。刻意催泪
示例:
  "他把粥端过来。碗沿上搁了一颗枣。她接过去的时候手指碰到他手背。
   他的手很热。她低头喝粥。那颗枣很甜。"
```

**0 常温 (Neutral)**
```
使用场景: 信息传递/过渡/环境建立/中性叙事段落
技法:
  - 叙述者保持中立距离
  - 情绪只通过角色行动暗示，不陈述
  - 句长适中，节奏平稳
  - 感官以视觉为主
禁止:
  - 情绪形容词
  - 叙述者评论
  - 过长描写（常温场景不能拖）
示例:
  "他走出门。街上没什么人。风灌进巷子，把地上的落叶卷起来。他往北走。"
```

**-1 微冷 (Cool)**
```
使用场景: 紧张气氛/对峙/悬疑/角色内心矛盾但未爆发
技法:
  - 退一步写。用外部动作和感官细节
  - 短句为主。连续短句后用一个中句释放
  - 对话极简——只传递必要信息
  - 感官: 触觉+听觉为主（冷感来自皮肤感受和声音的距离感）
禁止:
  - 情绪词
  - 角色的内心独白
  - 比喻（冷场景用比喻会"破功"——把读者拉出那个冷的感受）
示例:
  "他站在门口。手放在门框上。没有敲门。
   里面有声音。很轻。不像人。
   他把手放下来。转身。走了。
   走到巷子口的时候他停了。回头看。窗户是黑的。"
```

**-2 极寒 (Cold-Intense) — 余华模式**
```
使用场景: 情感高潮(死亡/告别/背叛/重大失去)/最痛的场景
技法:
  - 句长收缩到极限(6-12字)
  - 零情绪词。零评论。零内心独白
  - 零度对话标签（只说"他说"）
  - 全部用简单动词（站/看/走/摸/坐/放/拿）
  - 用感官缺席写情感在场
  - 重复一个无意义的动作（余华标志性重复结构）
  - 对话占比极低（5-15%）
禁止:
  - ANY情绪词
  - 比喻/夸张/排比
  - 叙述者评论
  - 角色解释自己的感受
  - 超过15字的句子
示例:
  "他把有庆放进坑里。
   土落下去。
   一下。两下。三下。
   他看着土把有庆盖住了。
   他站起来。
   腿有点麻。
   他拍了拍膝盖上的土。
   走了。"
```

## 二、温度切换规则

### 2.1 允许的切换

| 从 | 到 | 条件 |
|----|----|------|
| +1 | +2 | 有充分的情绪铺垫（≥500字的情感蓄积） |
| +2 | -1 | 高潮后的冷却（+2 → -1是最有效的冷热切换） |
| -2 | +1 | 极寒后的回暖（必须有一个缓冲——不能-2直接到+2） |
| 0 | -1 | 任何场景自然过渡 |
| -1 | -2 | 紧张升级到情感高潮 |

### 2.2 禁止的切换

| 从 | 到 | 原因 |
|----|----|------|
| +2 | +2 | 连续滚烫 = 情感透支。读者麻木 |
| -2 | -2 | 连续极寒 = 读者情感冻结。需要回暖 |
| 0 | +2 | 无铺垫的情感跳跃 |
| -2 | +2 | 极寒到滚烫 = 冰火两重天（除非是故意效果且铺垫充分） |
| 任何 | +2 | 连续使用超过1章 |

### 2.3 推荐切换频率

```
每章(3000-5000字)的标准温度曲线:

快档章节:  0 → +1 → +2 → -1 → -2 → 0
           开始→对抗升级→高潮→冷却→收束→过渡

中档章节:  +1 → 0 → +1 → 0 → -1 → +1
           日常→信息→冲突→缓冲→紧张→回暖

慢档章节:  +1 → +1 → +1 → 0 → -1 → +1
           持续的温热。有轻微起伏但不剧烈
```

## 三、温度衔接技法

### 3.1 渐进冷却 (+2 → +1 → 0 → -1)

用于从高潮自然过渡到收束：
```
+2: "她终于哭出来了。" (情绪释放)
+1: "哭了很久。嗓子哑了。眼睛肿了。" (情绪在消退，但身体感受还在)
0: "她洗了把脸。水很凉。" (用一个中性动作过渡)
-1: "镜子里的脸很陌生。她看了一会儿。把灯关了。" (冷下来的收束)
```

### 3.2 骤冷 (+2 → -1)

用于情感高潮后的留白：
```
+2: (激烈争吵达到顶点)
-1: "门关上了。屋子里只剩下风声。" (不写争吵的结果，直接跳到冷场)
```

### 3.3 冷中暖 (-1场景中的+1瞬间)

在冷的场景中插入一个微小的温暖细节，让冷更冷：
```
-1: 整场戏是冷酷的对峙
+1: 反派说了一句只有他们两个人懂的玩笑 (这比纯冷更让人心寒)
```

### 3.4 升温信号

从冷到热的过渡需要用这些信号铺垫：
```
信号1: 句长开始变长 (6-10字 → 12-18字)
信号2: 感官从冷感官(触觉/听觉)转向热感官(视觉/体感)
信号3: 对话比例开始上升
信号4: 角色的身体从"绷紧"变"放松"
```

## 四、场景类型推荐温度

| 场景类型 | 默认温度 | 可升温到 | 可降温到 | 注意事项 |
|---------|---------|---------|---------|---------|
| 角色日常互动 | +1 | +2 | 0 | 保持温热底。不要过热 |
| 角色内心反思 | +1 | +2 | 0 | 口语化。不要写成论文 |
| 核心对话冲突 | 0 | +1 | -1 | 对话温度 = 冲突强度 |
| 动作/打斗 | 0 | +1 | -1 | 动作本身是"热"的，但写法要"冷"(短句/无评论) |
| 环境建立 | 0 | +1 | -1 | 环境温度 = 本章基调 |
| 过渡 | 0 | — | — | 过渡保持常温 |
| 紧张/对峙 | -1 | 0 | -2 | 越紧张越冷 |
| 悬疑/未知威胁 | -1 | — | -2 | 用冷制造恐惧 |
| 情感高潮(正向) | +2 | — | +1 | 释放后快速退到+1 |
| 情感高潮(负向: 死亡/失去/背叛) | -2 | — | -1 | 越痛越冷 |

## 五、实战温度诊断

写作完成后执行：

```
Q1: 本章温度曲线是什么？
  → 画出5个关键节点的温度值

Q2: 有没有连续3个场景同一温度？
  → 有 → 插入一个温度变化

Q3: 情感高潮用对温度了吗？
  → 正向高潮(胜利/重逢/告白) → 应该是+2
  → 负向高潮(死亡/告别/背叛) → 应该是-2

Q4: 最冷的场景——有没有至少一个"冷中暖"细节？
  → 没有 → 插入一个。让冷更冷。

Q5: 温度切换有没有铺垫？
  → 检查每个切换点前后各200字
  → 如果没有升温/降温信号 → 补充
```

## 六、追加热成像：情绪-温度映射

不同情绪的最佳叙事温度：

| 情绪 | 推荐温度 | 原因 |
|------|---------|------|
| 狂喜 | +2 | 情绪完全打开 |
| 满足/幸福 | +1 | 温热。不夸张 |
| 平静 | 0 | 中性 |
| 不安/焦虑 | 0或-1 | 紧张但还没到冷 |
| 恐惧 | -1 | 冷制造恐惧 |
| 愤怒(爆发) | +2 | 热释放 |
| 愤怒(压抑) | -1 | 冷比热更有压迫感 |
| 悲伤(宣泄) | +2 | 允许哭 |
| 悲伤(深沉/永久失去) | -2 | 极寒。不哭但更痛 |
| 孤独 | -1或-2 | 冷的孤独比热的孤独更像孤独 |
| 希望 | +1 | 温热但不滚烫 |
| 绝望 | -2 | 最冷 |
| 怀念/思念 | +1 | 暖的怀念 |
| 麻木 | -2或-1 | 接近零度的情感 |
