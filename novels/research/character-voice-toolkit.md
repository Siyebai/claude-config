## 一、遮名可辨的七个维度

角色语音不只是"话多话少"。七个维度共同构成语音指纹：

| 维度 | 含义 | 示例对比 |
|------|------|---------|
| 句长偏好 | 说话用长句还是短句 | A: "走。" B: "我觉得我们可以考虑先离开这个地方。" |
| 词汇层级 | 用词的雅俗/文野/专业度 | A: "吃饭" B: "用膳" C: "干饭" |
| 句式习惯 | 陈述/疑问/反问/祈使的偏好 | A: 全是陈述句 B: 每三句一个反问 |
| 节奏模式 | 话是连绵的还是间断的 | A: 一口气说完 B: 说一句停一下 |
| 口头禅/惯用语 | 重复出现的词汇或句式 | A: 每句开头"你听我说" B: 每句结尾"是吧" |
| 回避语言 | 绝不说的话/绝不用的词 | A: 绝不说"爱" B: 绝不说"对不起" |
| 对话策略 | 直来直去/绕弯子/以退为进 | A: 想要什么直接说 B: 从不直接说要什么 |

## 二、同场景对比练习：20组

### 场景1: 有人踩了他们的脚

**角色A — 火爆直接型**:
"喂！你踩我脚了！"

**角色B — 阴阳怪气型**:
"不好意思啊，我的脚放错地方了。挡着您走路了。"

**角色C — 息事宁人型**:
(把脚挪开，没说话)

**角色D — 得理不饶人型**:
"踩到了。很疼。你得道歉。"

**角色E — 反讨好型**:
"没事没事！踩得好踩得好！我这鞋太旧了正好想换！"

### 场景2: 对方欠钱不还，催债

**A — 直接型**:
"上个月借你的二十两。该还了。"

**B — 绕弯子型**:
"最近手头怎么样？我最近也挺紧的。哎你说上个月那个事——"

**C — 威胁型**:
(不说话，只是站在门口。手放在刀柄上。)

**D — 感情绑架型**:
"当年你落魄的时候我是怎么帮你的？现在我有难了你就不管了？"

**E — 自嘲型**:
"我来要饭了。不是，我来要钱。不过跟要饭也差不多。"

### 场景3: 面对同伴的遗体

**A — 沉默行动型**:
(一声不吭。把遗体背起来。往前走。)

**B — 崩溃外放型**:
"起来！你他妈给我起来！你答应过——你答应过我的！"

**C — 理性克制型**:
"他的伤口在心脏位置。凶器是窄刃匕首。死亡时间不超过一个时辰。走不远。"

**D — 转向愤怒型**:
"谁干的。谁干的！我去杀了他。现在就去。"

**E — 仪式感型**:
"给他擦干净。把衣服整好。他活着的时候最在意体面。"

### 场景4: 向心上人表白

**A — 横冲直撞型**:
"我喜欢你。你喜欢我吗？喜欢就在一起。不喜欢就算了。"

**B — 百转千回型**:
"我昨天看见月亮特别好。然后就想到了你。也不是说你是月亮——就是——哎算了。"

**C — 行动替代型**:
(把一包东西塞给对方。转身就走。)"给你买的。"

**D — 逻辑论证型**:
"我想了很久。我们合适。性格互补。家境相当。你喜欢甜的我也喜欢。从各方面看都没问题。"

**E — 自毁型**:
"算了。你不会喜欢我的。我知道。我就是想说出来。你不用回答。"

### 场景5: 被人冤枉偷东西

**A — 暴怒自证型**:
"我没偷！你哪只眼看见我偷了？搜！随便搜！搜不出来你给我跪下！"

**B — 冷静讲理型**:
"第一，案发时我在城外。第二，这个锁不是撬开的。第三，我没有作案动机。"

**C — 不辩解型**:
"你们说是我就是我吧。"（转身走）

**D — 反向攻击型**:
"你丢了东西就赖我？你这么积极咬我，该不会是你自己偷的吧？"

**E — 委屈型**:
"真的不是我...求你们相信我...我真的没有..."

| 场景 | 角色A (火爆) | 角色B (冷静) | 角色C (沉默) | 角色D (幽默化解) | 角色E (低微讨好) |
|------|------------|-------------|-------------|----------------|----------------|

| 被困绝境无路可走 | "撞也要撞出一条路" | "未必。再想想。" (开始分析) | (坐下了。看着天。) | "死这儿也行。风景不错。就是缺点酒。" | "我们是不是要死在这里了..." |

## 三、同情绪不同表达：5情绪 × 4性格

### 愤怒

| 性格类型 | 身体反应 | 对话特征 | 典型行为 |
|---------|---------|---------|---------|
| 外放型 | 脸红/青筋/呼吸急促 | 直接发泄。音量升高。脏话 | 砸东西/打人/摔门 |
| 内敛型 | 下颌收紧/指节发白/呼吸变浅 | 话变少。每句变短。语气变冷 | 不说话/一个人待着/手上的活停下来 |
| 转移型 | 暂时的亢奋 | 笑着说的。但笑不是笑 | 用工作/运动/喝酒转移。不面对 |
| 计算型 | 表面无变化 | 语气变客气。但每句话都在埋钉子 | 开始记录/分析/计划报复。愤怒完全转化为策略 |

### 恐惧

| 性格类型 | 身体反应 | 对话特征 | 典型行为 |
|---------|---------|---------|---------|
| 外放型 | 出汗/发抖/脸色发白 | 音量变大（用声音盖住恐惧） | 逃跑/躲藏/尖叫 |
| 内敛型 | 手心出汗/呼吸变浅/瞳孔放大 | 不说话。嘴唇发干 | 站在原地不动。盯着威胁源 |
| 亢奋型 | 肾上腺素飙升。眼睛发亮 | 话变多。语速变快。内容跳跃 | 反而往前走。恐惧转化为刺激 |
| 麻痹型 | 身体僵硬/四肢发冷/大脑空白 | 说不出话来 | 一动不动。需要别人拽着走 |

### 悲伤

| 性格类型 | 身体反应 | 对话特征 | 典型行为 |
|---------|---------|---------|---------|
| 宣泄型 | 哭/哽咽/身体蜷缩 | 断断续续。说不了完整的句子 | 找没人的地方放声哭 |
| 压抑型 | 嗓子堵/胸口闷/眼睛发涩但不哭 | 话变多（用说填满以免哭出来） | 不停做事。不让自己停下来 |
| 隔离型 | 身体感受变钝。什么都感觉不到 | 不说话。把自己关起来 | 一个人待着。不接任何话 |
| 攻击型 | 焦虑转化为愤怒 | 找茬吵架。说话带刺 | 把悲伤转成愤怒发泄给身边的人 |

### 喜悦

| 性格类型 | 身体反应 | 对话特征 | 典型行为 |
|---------|---------|---------|---------|
| 分享型 | 眼睛发亮/能量充沛/坐不住 | 话变多。到处说 | 打电话/奔走相告/请客 |
| 内敛型 | 嘴角动了一下。眼神变暖 | 说话比平时多了一点 | 自己坐在那儿回味 |
| 怀疑型 | 先看是不是搞错了。不敢相信 | "真的？你确定？" | 反复确认。不敢高兴得太早 |
| 压抑型 | 高兴但不允许自己太高兴 | "还行。" 但嘴角压不住 | 表面淡定。转身之后跳起来 |

### 撒谎时

| 性格类型 | 身体反应 | 对话特征 | 典型行为 |
|---------|---------|---------|---------|
| 紧张型 | 眼神飘忽/摸鼻子/搓手指 | 话变多。解释过多。细节过多 | 补太多细节反而暴露 |
| 冷静型 | 完全无变化。眼神反而更坚定 | 话变少。只用"是/不是/没有" | 不解释。解释就是心虚 |
| 反向型 | 故意显得特别放松 | 语气比平时更随意。开着玩笑撒谎 | 用笑声和玩笑掩盖 |
| 装傻型 | 表情迷茫"你在说什么" | 装听不懂。绕圈子。反问对方 | 不正面回答任何问题 |

## 四、自动化语音检测规则

以下规则可集成到 `novel style lint` 或独立工具：

### 规则1: 对话标签分布检测
```
检测: 每个角色的对话标签类型分布
预期: 不同角色的标签模式应有差异
  - 角色A: 80%无标签 + 15%动作打断 + 5%"说"
  - 角色B: 60%"说" + 30%动作打断 + 10%无标签
警告: 两个角色标签分布相似度 > 70% → 语音区分度不足
```

### 规则2: 句长分布检测
```
检测: 每个角色说的话的平均句长 + 标准差
  - 提取章节中每个角色所有对话
  - 计算每人对话的平均句长和标准差
  - 两两对比: 如果两个角色句长偏差 < 15% → 标记
```

### 规则3: 高频词差异检测
```
检测: 每个角色对话中的Top-10高频实词
  - 如果两个角色共享 > 7/10 高频词 → 语音同质化
  - 特别关注: 口头禅/语气词/惯用句式
```

### 规则4: 回应模式检测
```
检测: 面对同一挑衅/提问，不同角色的回应模式
  - 如果回应模式相同（都是直接回应/都是回避/都是反问）→ 标记
  - 每个角色应有其独特的对话策略
```

### 规则5: 情绪表达一致性
```
检测: 同一角色在不同章节中同一情绪的触发阈值和表达方式
  - 如果该角色在前一章愤怒时沉默，本章愤怒时大吼大叫 → 除非有弧线推进，否则标记
```

## 五、角色语音设计模板 (快速版)

```yaml
角色名:
一句话声音定位:   # 例如"话很少但每句都让人忘不掉"
句长偏好:         # 极短(1-5字)/短(5-10字)/中(10-20字)/长(20+字)

惯用句式 (3个):
  - "..."         # 这个角色最常说的话的结构
  - "..."
  - "..."
一句封神:         # 最能代表这个角色的一句台词
绝不能说的话:      # 这个角色说出这句话 = OOC
激动时:           # 话变多/话变少/语速变快/语速变慢/干脆沉默
紧张时:           # 口头禅频率增加/开始绕弯子/直接不说话/开始讲冷笑话
```

## 六、硬规则速查

### 必须做到
- 每个有名字的角色一张语音卡
- 每写完一场对话 → 遮名重读
- 每个角色的第一句话必须建立声音印象
- 对话中的人物语言差异 ≥ 3个维度
- 配角对话 ≠ 主角对话的降级版（配角的语音是独立的，不是"没那么精彩的主角语音"）

### 绝对禁止
- 两个角色可以互换台词而读者察觉不到
- 对话标签带情绪副词（"他冷冷地说"）
- 角色在自己的对话中说"我生气了""我很难过"
- 所有人都用同样的句式结构说话
- 角色在对话中充当作者的传声筒（用角色的嘴说作者想表达的道理）
